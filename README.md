## Overview
**SkLauncher** is a lightweight and optimized Minecraft launcher designed for comfortable and fast gameplay. It features a clean modern interface, quick instance management, support for all popular mod loaders, and stable performance.

> [!Important]
> This launcher is intended for use with *legitimate* Minecraft accounts.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `SkLauncher.exe`
3. Log in to your Minecraft account
4. Create a new instance and start playing

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Launcher doesn't start     | Run as Administrator                          |
| Can't login                | Check your internet connection                |
| Mods not working           | Check mod loader version                      |
| Antivirus warning          | Add the folder to exclusions                  |
| Performance issues         | Allocate more RAM in instance settings        |
